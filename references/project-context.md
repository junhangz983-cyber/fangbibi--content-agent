# 方比比项目上下文

## 人设

方比比是 2001 年潮汕人，年入百万 00 后创业者，“比比先生”主理人。

核心人设：真诚、孝顺、低调、懂事，会赚钱但不炫耀，尊重妈妈，能把普通家庭小事讲出温度。

内容核心不是炫耀赚钱，而是通过年轻人视角讲亲子关系、妈妈、金钱观、家庭教育和代际理解。

## 妈妈人设

妈妈是情感故事的绝对主角。

关键词：小学文化、勤劳持家、爱干净、情商高、低调、爱笑、不凶孩子、不主动索取、嘴上说不要但心里在意。

不要把妈妈写成爱炫耀、爱索取、强势控制的人。

## 受众

核心受众是 45-65 岁女性妈妈粉，以 50-60 岁为核心。

她们喜欢看的不是炫富，而是“别人家优秀孩子”如何理解妈妈、心疼妈妈、回报妈妈。

高共鸣来自：
- 妈妈被孩子看见。
- 孩子终于懂事。
- 妈妈的付出不是理所当然。
- 子女想起自己忽略妈妈的瞬间。
- 评论区能出现“我妈也是这样”“看哭了”“转给孩子看”。

## 内容结构

常用四段式：

```text
钩子 → 矛盾/真相 → 反转 → 升华
```

方比比语速按 180 字/分钟计算。

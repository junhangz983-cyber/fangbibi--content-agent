# 小号老板 Vlog 标题手册

本文件内置 08 手册的可执行标题逻辑。生成方比比小号老板 Vlog 标题时以本文件为主，不依赖外部项目路径。

## 适用范围

用于方比比小号、老板日常、员工视角、团队、会议、大会、线下课、出差、公司、创业、网红老板、00 后老板等 Vlog 内容。

小号老板 Vlog 与大号情感口播是两条线：

- 大号情感口播：服务妈妈粉，常用 `年入百万00后 + 钩子`。
- 小号老板 Vlog：服务同龄人和泛流量，主打慕强、反差、猎奇、老板人设宽度，不套固定句式。

## 核心公式

```text
人物关系 + 事件 + 冲突 + 悬念 = 合格标题
```

标题必须先回答四个问题：

- 谁在看谁：员工看老板、团队看老板、外人看 00 后老板。
- 发生了什么：大会、线下课、出差、开会、改课件、请吃饭。
- 哪里反常：老板给员工当牛马、员工休息老板干活、00 后老板扛全场。
- 为什么想点：这一天发生了什么，老板到底怎么做，结果先不要讲完。

## 受众与情绪

核心受众：

- 20-35 岁同龄人。
- 对年轻老板、网红老板、创业团队、公司日常好奇的泛流量。

主情绪：

- 慕强：年轻但能扛事。
- 反差：老板不是高高在上，而是在现场干活。
- 猎奇：00 后老板的工作日到底是什么样。
- 爽感：团队被带动，事情被推进，但不炫富。

## 调性边界

可以：

- 更外放。
- 更网感。
- 更像 Vlog 标题。
- 使用轻度口语和梗。

不能：

- 破坏真诚低调底色。
- 炫富、装成功、端着教育别人。
- 攻击员工、攻击同行、攻击质疑者。
- 堆标签但没事件。
- 泄露结果，让观众没有点击理由。

## 关键词矩阵

直接关键词：

- 00 后老板
- 网红老板
- 员工视角
- 老板日常
- 公司
- 团队

场景关键词：

- 百人大会
- 线下课
- 出差
- 改课件
- 开会
- 带团队
- 请全公司吃饭

冲突关键词：

- 给员工当牛马
- 员工下班老板还在干
- 年轻老板扛全场
- 出差回来继续改
- 老板比员工还忙

反差关键词：

- 00 后
- 网红
- 老板
- 当牛马
- 的一天
- 嘎嘎忙

## 网感词边界

可用但要克制：

- 牛马
- 嘎嘎
- 宝
- 的一天

慎用或不用：

- 沉浸式
- 凡尔赛
- 绝绝子
- yyds

这些词如果不能增强人物关系、事件、冲突、悬念，就不要硬塞。

## 好标题标准

好标题通常具备：

- 有人物关系：不是“开大会”，而是“00 后老板给员工当牛马”。
- 有具体事件：不是“忙碌一天”，而是“办百人大会”“出差回来改课件”。
- 有反差冲突：不是“老板很努力”，而是“员工休息老板还在干”。
- 有悬念：不直接写圆满成功，不提前讲结果。

## 反例

- `00后老板沉浸式体验`
  - 问题：自我视角，没事件，没冲突，没悬念。

- `轻松搞一场百人线下大会`
  - 问题：泄露结果，削弱冲突，没有网感。

- `网红老板创业日常`
  - 问题：标签堆叠，太泛。

## 可复用方向

- `00后网红老板举办百人大会 给员工当牛马的一天`
- `员工下班后 00 后老板还在改课件`
- `00后老板出差回来 第一件事竟然是给团队补课`
- `员工视角看网红老板开一场百人大会`

使用这些方向时只复刻机制，不机械复制。

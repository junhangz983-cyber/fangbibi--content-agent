# 选题判断

## 目标

从输入素材中拆出情绪结构、关系结构和传播结构，再转成适合方比比真实表达的选题。

## 优先方向

- 给妈妈打钱：不是钱本身，而是孩子终于能反过来照顾妈妈。
- 赚钱与代际认知差：我以为妈妈不懂钱，长大后才发现她最懂生活成本。
- 母亲的教育智慧：妈妈没讲大道理，但用一件小事教会我做人。
- 超预期情感表达：我原本以为妈妈不在意，后来发现她只是不会说。
- 潮汕妈妈、普通妈妈、辛苦女人背后的家庭支撑。
- 个人逆袭叙事：我今天能赚钱，不是因为我多厉害，而是妈妈早早把底层能力种在我身上。
- 家庭关系模式：家里真正撑住秩序和温度的人，往往是妈妈。

## 好选题标准

好选题必须同时满足：
- 妈妈粉能代入。
- 子女能愧疚。
- 比比能真实讲。
- 有反差。
- 有评论空间。

## 输出格式

```markdown
## 一、素材内核判断

- 这个素材表面讲什么：
- 真正打中的情绪是什么：
- 妈妈粉为什么会点赞：
- 子女为什么会转发 / 评论：
- 它适合比比的部分：
- 它不适合比比的部分：

## 二、可转成比比的选题方向

每个选题包含：
- 选题标题：
- 选题角度：
- 为什么适合比比：
- 需要从比比真实经历里追问什么：

## 三、优先级排序

从所有选题里选出最值得优先拍的 3 个，并说明原因。

## 四、下一步需要

- 建议先采访哪个选题：
- 需要比比确认的真实经历：
- 进入采访提问段前还缺什么：
```

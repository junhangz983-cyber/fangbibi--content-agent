# 方比比情感口播标题补丁

> 本文件是 `fang-bibi-content-agent/references/title-generation.md` 的情感口播标题补丁。
> 情感口播标题不单独 Skill 化，继续在方比比内容 Agent 内执行；本补丁负责把本地 06 逻辑手册作为主依据。

## 一、适用场景

当用户提供的是妈妈、母子、亲情、红包、礼物、花钱、低调家风、潮汕妈妈、孩子长大等素材时，判断为情感口播标题。

## 二、优先依据

处理前优先读取：

- `references/title-generation-emotional-handbook.md`
- `fang-bibi-content-agent/references/title-generation.md`

不要依赖外部项目路径。外部 `IP手册_06_爆款标题创作.md` 只能作为当前项目内的增强资料，不是调用本 skill 的前置条件。

## 三、核心规则

- 默认保留 `年入百万00后 + 钩子`。
- 主标题负责点击，不直接讲答案。
- 副标题、封面小字、收尾金句负责补情绪。
- 优先抓妈妈粉会点赞转发的关系张力：妈妈 + 钱、妈妈 + 礼物、妈妈 + 低调、妈妈 + 家风、妈妈 + 孩子长大。
- 标题必须正向、温暖、治愈，但可以有问题感和反差。

## 四、video-title-gen 的使用边界

`video-title-gen` 只作为通用标题能力补充，可以借用：

- 悬念型
- 数字型
- 情感型
- 痛点型
- 对比型

但不能覆盖 06 手册，不能裸用通用模板。

## 五、执行提醒

生成标题前先判断：

1. 这是主标题还是副标题？
2. 有没有提前讲答案？
3. 有没有具体物件、数字、动作或关系？
4. 妈妈粉看到后会不会觉得“这和我有关”？
5. 是否保留 `年入百万00后` IP 锚点？

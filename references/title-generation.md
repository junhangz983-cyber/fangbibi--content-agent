## 顶级原则

标题的唯一评判标准：观众会不会点击，会不会转发。

标题不是总结文案，而是制造点击理由。它应该抓住妈妈粉心里的问题，而不是直接泄露文案里的答案。

以下四条是方比比标题的**顶级原则**，所有标题创作、评估、优化都必须以此为准绳：

### 1. 高情绪浓度

标题必须让妈妈粉在 1 秒内感受到强烈的情绪波动。情绪要浓、要扎心、要现实、要急迫。可以是愧疚、心疼、温暖、骄傲，但不能是恐惧、焦虑、指责。单一情绪足够浓 > 多重情绪但分散。

### 1.1 优先级说明

**标题创作的第一优先级是情绪浓度。** 画面感、冲突感、信息密度不是并列目标，而是服务情绪浓度的三种抓手。

- 画面感：让观众一眼看见场景。
- 冲突感：让观众感到关系里的反常和张力。
- 信息密度：让观众知道这里有具体事件、动作、数字或物件。

如果一个标题信息很多但不够有情绪，就继续换画面、换冲突、换关系，不要先追求完整概括。

### 2. 高信息浓度

标题必须一句话讲完核心关系。用具体数字、具体动作、具体物件承载信息，拒绝空泛表达。例如“妈妈300块衣服都嫌贵” > “妈妈很舍不得花钱”。

### 3. 高画面感

标题必须有可视化的画面，让观众脑海中自动浮现场景。抓动作不抓状态，抓画面不抓评价。例如“妈妈用身体挡在我面前” > “妈妈很保护我”。

### 4. 展现过程而非结果

标题是未闭合的环。绝不直接在标题里告诉观众答案。展现“这件事怎么发生的”，不展示“最后怎么样了”。例如“被妈妈发现赚太多钱后” > “我很低调不炫耀”。

## 创作原则

- 绝不泄露结果，标题必须开环。
- 优先抓观众心里的问题，不抓作者已经说出的金句。
- 用钱、数字、具体动作反向表达抽象情感。
- 用“只 / 唯一 / 都 / 每一个”等极限词制造稀缺张力。
- 多用具体场景和具体数字，不要空泛表达。
- 标题要正向、温暖、治愈，避免恐惧、指责、焦虑。
- 不要爹味说教，不要婚恋话题，不要让父亲抢戏。
- 每个标题都要让人想知道：然后呢？为什么？他怎么做的？

## 评分维度

每个标题按 1-5 分评分：

| 维度 | 说明 |
|---|---|
| 妈妈粉代入感 | 50-60 岁妈妈看到后是否觉得和自己有关 |
| 点击悬念 | 是否产生“为什么 / 然后呢”的缺口 |
| 比比人设贴合度 | 是否真诚、低调、孝顺，不炫富 |
| 具体度 | 是否有钱、动作、人物关系、具体场景 |
| 情绪浓度 | 是否有愧疚、心疼、被看见、被反哺 |
| 开环程度 | 是否没有把答案提前说完 |
| 红线风险 | 是否避开婚恋、爹味、爸爸抢戏、恐惧感 |

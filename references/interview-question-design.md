# 采访提问设计

## 核心原则

采访提问不是替比比编故事，而是把真实经历里的画面、动作、原话、情绪转折和金句问出来。

提问分三层：

1. 前置提问：打开表达欲，让比比进入记忆现场。
2. 细节追问：挖具体画面、动作、原话、时间点。
3. 收束提问：提炼情绪变化、认知反转和可用金句。

## 前置提问规则

前置提问不是为了马上拿答案，而是为了让比比想讲。

必须：
- 开放式，不用“有没有 / 是不是 / 会不会 / 你觉得吗”。
- 不直接说出选题结论。
- 不把答案塞给比比。
- 优先问生活画面、具体时刻、妈妈动作、妈妈原话。
- 让比比先回到记忆里，而不是进入观点表达。
- 问题要轻，不能像审问，也不能像命题作文。

优先入口：
- 小时候最常见的一个画面。
- 妈妈最常说的一句话。
- 某次花钱、收钱、拒绝礼物、舍不得买东西的场景。
- 第一次意识到自己可以反过来照顾妈妈的时刻。
- 妈妈嘴上说不要，但动作很诚实的细节。

错误问法：
- 你妈妈是不是很辛苦？
- 你有没有给妈妈打过钱？
- 你是不是赚钱以后更理解妈妈了？
- 你觉得妈妈伟大吗？
- 你妈妈会不会嘴上说不要其实心里想要？

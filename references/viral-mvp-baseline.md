# 方比比爆款评分基准 MVP

来源：飞书知识库《视频号&抖音号数据》，包含视频号与抖音号爆款样本的标题、播放、爱心和文案。

本文件只用于发布前流量自检的参照，不用于承诺播放量预测。

## 样本能力边界

已具备：

- 爆款标题 / 选题名。
- 播放量。
- 爱心数。
- 文案 / 逐字稿。
- 视频号与抖音号平台区分。

暂缺：

- 普通样本和没跑出来样本。
- 转发、评论、收藏、涨粉、完播率、3 秒留存。
- 发布时间与 T+3d 统一口径。
- 封面标题、发布节奏、评论区高赞原话。

因此本基准只支持：

- 判断新稿是否“像方比比过往爆款”。
- 提炼爆款母体、情绪结构、标题开环、具体场景。
- 辅助 8 维发布前评分。

不支持：

- 稳定预测播放量。
- 判断平台分发概率。
- 精准比较点赞率、转发率、完播率。

## 爆款母体

### 1. 妈妈存在本身就是优点

代表样本：

- 视频号《最喜欢妈妈哪一点》：播放 1392w，爱心 45.3w。
- 抖音《最喜欢妈妈哪一点》：播放 550.7w，爱心 13.1w。

爆点：

- 开头问题极简：你妈妈最大的优点是什么？
- 回答不讲具体功能，而是把妈妈本人推到最高价值。
- 情绪是“妈妈被孩子看见”，妈妈粉会获得强确认感。

评分启发：

- 如果新稿能让妈妈觉得“我这个人本身就值得被爱”，妈妈粉代入和亲情投射加分。
- 但不能泛泛夸妈妈伟大，必须有具体表达或反差。

### 2. 给妈妈打钱不是钱，是反向照顾

代表样本：

- 视频号《给父母打多少钱才够》：播放 1363w，爱心 12.2w。
- 视频号《不敢给妈妈打钱》：播放 656.5w，爱心 19.2w。
- 抖音《不敢给妈妈打钱》：播放 519w，爱心 8.4w。
- 抖音《到底给妈妈多大红包》：播放 283.1w，爱心 4.5w。

爆点：

- 钱是入口，真实情绪是“孩子终于有能力照顾妈妈”。
- 妈妈嘴上不要，行动里全是替孩子攒着。
- 具体物件强：520、年终奖、私房钱、备忘录、红包。

评分启发：

- 有具体金额、理由、妈妈收钱动作、妈妈后续反应时，真实细节密度加分。
- 如果只讲“我要孝顺妈妈”，不开钱和具体行为，传播力弱。

### 3. 赚钱后不敢告诉父母

代表样本：

- 视频号《赚了钱不敢告诉父母》：播放 628.5w，爱心 10.1w。
- 抖音《赚了钱不敢告诉父母》：播放 308.8w，爱心 6.4w。
- 视频号《被妈妈揭穿赚太多钱》：播放 1155.3w / 1178.4w，爱心 10.7w。

爆点：

- 反常识：赚到钱反而不敢说。
- 代际认知差：父母不一定懂工作，但希望孩子平安。
- 家风高光：不张扬、不炫耀、自己开心就好。

评分启发：

- “比比现在有结果了，但家里关系反而更安静/更客气/更不张扬”是高价值切面。
- 适合测试“火了以后、赚钱以后、当老板以后”的关系变化。

### 4. 妈妈只教了一件事

代表样本：

- 视频号《妈妈只做了一件事》：播放 526.9w，爱心 10.2w。
- 视频号《母亲只教了一件事》：播放 542.6w，爱心 11.3w。
- 抖音《母亲只教了一件事》：播放 1173.3w，爱心 26w。

爆点：

- 低门槛捷径：妈妈只做了一件事，却影响孩子一生。
- 教育智慧来自具体生活，不是说教理论。
- “夸孩子”“低调攒钱”“厚积薄发”都能挂到孩子今天的结果。

评分启发：

- 如果新稿能从比比一个品质反推妈妈当年的一个动作，亲情投射和情绪反差加分。
- 如果变成教育方法论，容易失去方比比的温度。

### 5. 潮汕女人 / 妈妈撑住家庭秩序

代表样本：

- 视频号《潮汕女人都很招财》：播放 592w，爱心 7.8w。
- 抖音《潮汕女人都很招财》：播放 592w，爱心 5.6w。

爆点：

- 地域标签带认同：潮汕、勤快、爱干净、招财。
- 妈妈不是抽象伟大，而是把家里秩序、卫生、待人处世都撑起来。
- 具体动作强：客厅房间一天打扫两遍、茶具碗筷立马洗。

评分启发：

- “妈妈每天做什么”比“妈妈是什么样的人”更能出选题。
- 爱干净、勤快、会做人、爱笑这类品质，要通过细节问出来。

### 6. 个人逆袭与家庭教育

代表样本：

- 视频号《被父母劝高三复读》：播放 707.7w，爱心 8.3w。
- 抖音《被父母劝高三复读》：播放 351.1w，爱心 4.9w。
- 抖音《男生和女生谁更赚钱》：播放 391.3w，爱心 7.9w。

爆点：

- 年入百万 00 后的人设能带动“为什么他能做到”。
- 观众想知道野心、机会、能力从哪里来。
- 但方比比的逆袭叙事要回到低调、内敛、敢争取、妈妈教育，不要变炫耀。

评分启发：

- 老板/赚钱/创业类内容如果能回到家风和妈妈教给他的底层能力，安全性更高。
- 只讲成功经历，妈妈粉代入会变弱。

### 7. 家庭关系真相

代表样本：

- 抖音《孩子不尊重妈的真相》：播放 375.3w，爱心 5.3w。

爆点：

- 用“真相”打开家庭隐痛。
- 评论空间强，容易引发妈妈粉代入。

风险：

- father 容易抢戏。
- 家庭冲突容易变重，偏离方比比低调温暖气质。

评分启发：

- 可借鉴“关系真相”的结构，但少用父亲冲突，多回到妈妈动作和比比理解。

## 8 维评分时的参照口径

### 妈妈粉代入

高分信号：

- 妈妈能觉得“这就是我”。
- 讲的是妈妈日常被忽略的付出。
- 不是夸大牺牲，而是具体动作被看见。

低分信号：

- 只讲比比成功。
- 只讲抽象孝顺。
- 妈妈只是背景。

### 亲情投射强度

高分信号：

- 孩子反过来照顾妈妈。
- 妈妈嘴硬但爱孩子。
- 小时候不懂，长大才懂。

低分信号：

- 情绪只停在观点。
- 没有关系变化。

### 开头点击钩子

高分信号：

- 反常识：赚钱了反而不敢说，不敢给妈妈打钱。
- 极简问题：妈妈最大的优点是什么？
- 关系变化：妈妈开始不敢打扰你、开始把你当大人。

低分信号：

- 开头就是结论。
- 开头太像鸡汤。

### 情绪反差

高分信号：

- 嘴上不要，心里在意。
- 以为妈妈不懂，后来发现她最懂。
- 以为给钱是爱，后来发现妈妈要的是陪伴和放心。

低分信号：

- 全程平铺直叙。
- 只有夸，没有反转。

### 真实细节密度

高分信号：

- 金额、红包、520、备忘录、牛肉丸、茶具、碗筷、房间、签字、成绩单。
- 妈妈原话和动作。

低分信号：

- 没有物件，没有场景，没有原话。

### 标题开环

高分信号：

- 给妈妈打多少钱才够。
- 不敢给妈妈打钱。
- 妈妈只做了一件事。
- 赚了钱不敢告诉父母。

低分信号：

- 标题直接把结论说完。
- 标题像总结，不像问题。

### 人设安全

高分信号：

- 低调、不张扬、真诚、孝顺、尊重妈妈。
- 赚钱是背景，家庭关系才是主体。

低分信号：

- 炫富。
- 教训别人。
- 妈妈显得索取或炫耀。

### 转发理由

高分信号：

- 妈妈想转给孩子看。
- 子女想转给兄弟姐妹或朋友。
- 内容替妈妈说出没说出口的话。

低分信号：

- 只适合点赞，不适合转发。
- 只有个人故事，没有关系投射。

## MVP 使用方式

发布前自检时，先读本文件，再按 `traffic-check.md` 的 8 维评分输出。

判断逻辑：

```text
新稿是否具备过往爆款的母体
-> 是否有新的关系变化或场景切面
-> 是否有具体物件/动作/原话
-> 是否有开头问题和标题开环
-> 是否值得发布或需要修改
```

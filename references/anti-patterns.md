# 方比比内容反例黑名单

本文件是全流程黑名单。处理任何方比比任务时，先用它做底线过滤；具体阶段规则仍读取对应 reference。

## 全流程禁止

| 反模式 | 为什么错 | 正确处理 |
|---|---|---|
| 替比比编真实经历 | 会制造虚假人设，后续采访和剪辑都无法落地 | 停在当前阶段，列出需要比比确认的问题 |
| 为了完整强行跑全流程 | 会跳过真实素材，直接产出假内容 | 只做当前材料能支持的最早阶段 |
| 把妈妈写成索取、控制、攀比 | 破坏妈妈粉的情感认同 | 写妈妈的低调、体面、嘴硬心软、托举孩子 |
| 把比比写成炫富或教育别人 | 偏离真诚低调人设 | 年入百万只做 IP 锚点，落点放在理解妈妈 |
| 用抽象好话替代具体细节 | 内容会像鸡汤，缺少可剪画面 | 回到人物动作、原话、物件、数字、场景 |
| 照搬对标视频 | 只复制表层标题，不能变成方比比选题 | 拆内核，再找比比自己的真实经历 |

## 选题段禁止

- 不直接给标题、封面、完整脚本。
- 不把“别人爆过”当作“比比能拍”的理由。
- 没有比比真实经历时，不进入采访提问之后的阶段。

## 采访提问段禁止

- 不问“有没有 / 是不是 / 会不会”这类限制性问题。
- 不把结论喂给比比。
- 不用问题诱导比比证明预设观点。

## 重构段禁止

- 不补连接句、总结句、升华句。
- 不把意思相近当作逐字逐句一致。
- 不跨 take 合并成原文不存在的新句子。
- 不一刀切删除能推动叙事、增强真实感的编导提问。

## 标题段禁止

- 不提前泄露正文答案。
- 不把正文金句、封面小字、收尾升华当主标题。
- 不用通用标题模板覆盖方比比人设。
- 不把情感口播和小号老板 Vlog 的标题逻辑混用。

## 发布前自检段禁止

- 不把主观“我觉得会爆”当作预测结论。
- 不跳过标题开环、人设安全、妈妈粉代入、转发理由。
- 不在没有最终标题和最终稿时进入发布前流量自检。

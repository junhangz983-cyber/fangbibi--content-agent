# 标题案例校准与 Check

本文件承接标题段的历史共创经验、案例库校准和题材专项 check。标题生成先读 `title-generation.md` 的通用规则，再按需要读取本文件做校准。

## 历史 Check 补充

后续标题 check 优先追加到本文件，不改动 `SKILL.md` 和 `title-generation.md` 的主流程。本文件是标题段的执行前后检查区：先按流程推理，再生成标题，再用案例校准。

### 1. 标题任务优先级

标题的第一任务是让观众点进来。标题不是“准确概括全文”，而是从文稿里提炼最能制造好奇心的入口。

优先级：

1. 能不能让观众好奇，想点进去。
2. 标题和正文是否能接得上。
3. 是否同时具备情绪浓度和信息密度。
4. 是否符合方比比人设和红线。
5. 是否总结得完整。

注意：第 5 点不是标题的核心目标。标题不需要完整，它需要开口子。

判断标题时先问：

- 观众看到后会不会想问“为什么”？
- 观众会不会想知道“然后呢”？
- 观众会不会想确认“这句话到底什么意思”？
- 这个标题有没有一个明确但没说透的缺口？

可以接受标题只抓文稿中最有点击价值的一个点，而不是覆盖全部内容。

### 2. 标题推理流程

不要直接开始写 10 个标题。标题生成前必须先完成推理：

1. 表面内容：这条文稿表面讲什么。
2. 核心爆点：最值得观众点开的事件、动作、反差、数字或物件。
3. 隐藏情绪：文稿没有明说，但妈妈粉能感受到的情绪。
4. 点击缺口：观众点进来想确认什么。
5. 信息锚点：哪个具体词、数字、动作或物件最适合放进标题。
6. 可用模型：这条文稿适合哪几类标题入口。
7. 候选标题：按模型生成候选。
8. 最终排序：按点击好奇心优先排序。

标题要优先抓“核心爆点 + 隐藏情绪 + 点击缺口”的交叉点，不要只按正文总结。

### 3. 情绪浓度与信息密度

所有标题都必须同时具备：

- 情绪浓度：让妈妈粉感到被看见、被心疼、被孩子放在心上，或看到孩子懂事长大的瞬间。
- 信息密度：让观众一眼知道这条内容里有一个具体事件、关系、动作、数字、物件或反差值得点进去。

只满足一个不够：

- 只有情绪、没有信息：容易变成空泛鸡汤。例如“妈妈真的很伟大”。
- 只有信息、没有情绪：容易像流水账。例如“第一次给妈妈发红包”。

合格标题公式：

```text
具体信息点 + 隐藏情绪 + 开环悬念
```

检查问题：

- 有没有具体信息点？如钱、礼物、红包、表情包、衣服、三个包、一句话、一个动作。
- 有没有情绪张力？如嘴上说不要、最爱、舍不得、放心、孩子长大、妈妈被放在心上。
- 信息点和情绪点是否绑定在同一个关系里？
- 看完标题后，观众是否既知道“这条有东西”，又想知道“然后呢”？

### 4. 高共鸣关系与禁用词

优先使用这些高共鸣关系：

- 妈妈 + 钱
- 妈妈 + 礼物
- 妈妈 + 低调
- 妈妈 + 家风
- 妈妈 + 孩子长大

优先标题方向：

- 妈妈舍不得花钱怎么办
- 赚多少钱才够养好妈妈
- 妈妈为什么总先省自己
- 怎么让妈妈放心花钱
- 妈妈发现我赚太多钱后
- 妈妈知道我火了以后
- 妈妈教我的低调智慧
- 给妈妈多少钱才够
- 妈妈最想要的到底是什么
- 孩子长大后怎么对妈妈

不要使用：

- “秘籍大公开”
- “指南”
- “沉浸式体验”
- 爹味说教表达
- 婚恋话题
- 让父亲抢戏的表达

### 5. 常用标题模型

每条文稿至少尝试 4 类标题模型，再挑最有点击好奇心的方向。

| 模型 | 作用 | 示例 |
|---|---|---|
| 反事实型 | 用“如果没有……”制造强假设和悬念 | 如果没有妈妈 / 如果妈妈没生下我 |
| 关系定位型 | 重新定义妈妈在家庭里的位置 | 妈妈才是家里的主心骨 |
| 具体责任型 | 用数字、动作、责任重量承载情绪 | 妈妈扛起一家四口 |
| 情绪归因型 | 把孩子的优秀归因到妈妈身上 | 孩子优秀不是偶然的 |
| 具体物件型 | 用红包、表情包、衣服、礼物等物件制造入口 | 妈妈最爱的三个包 |
| 反常动作型 | 抓妈妈嘴上说不要、身体很诚实的反差 | 妈妈说不要红包 |
| 超预期回答型 | 提问后给出跳出常规的答案 | 最喜欢妈妈哪一点 |
| 量化孝心型 | 用钱、数字、红包衡量无法衡量的爱 | 给父母打多少钱才够 |
| 认知倒置型 | 成功后出现反常行为 | 不敢给妈妈打钱 / 赚了钱不敢告诉父母 |
| 家庭博弈型 | 家人发现成功后反而低调克制 | 被妈妈揭穿赚太多钱 |
| 地域认同型 | 用潮汕妈妈/女人激发身份认同 | 潮汕女人都很招财 |

### 7. 已定标题案例库

案例一：第一次给妈妈发红包，妈妈嘴上说“不要”“老妈有钱”，领完后发可爱表情包，形成母子默契。

最终定题：

```text
年入百万00后，妈妈最爱的三个包
```

校准要点：

- 不直接写“红包”或“表情包”，而是用“三个包”制造悬念。
- “最爱”是温暖正向词，符合妈妈粉可接受情绪。
- 信息密度：三个包，具体数量和物件。
- 情绪浓度：妈妈最爱，暗含孩子把妈妈放在心上。
- 开环悬念：三个包到底是什么，为什么是妈妈最爱。
- 这类标题比“妈妈说不要红包”更有包装感，也更适合点击。

已验证模型校准：

- `妈妈最爱的三个包`：具体物件型，信息点是“三个包”，隐藏情绪是“妈妈被孩子放在心上”。
- `如果没有妈妈`：反事实型，点击缺口是“没有妈妈会怎样”。
- `妈妈才是家里的主心骨`：关系定位型，转发认同强。
- `妈妈扛起一家四口`：具体责任型，信息密度强。

### 8. 爆款案例库校准

标题生成时要参考本文件内置爆款案例库校准，不复制旧标题，但复刻其点击机制。

已验证高互动机制优先级：

| 优先级 | 已验证标题 | 数据表现 | 可复刻机制 |
|---:|---|---|---|
| 1 | 最喜欢妈妈哪一点 | 视频号 1392w，互动率 3.26% | 提问 + 限制条件 + 超预期回答 |
| 2 | 不敢给妈妈打钱 | 视频号 656w，互动率 2.92% | 时间线 + 具体数字 + 情感转折 |
| 3 | 母亲只教了一件事 | 抖音 1173w，互动率 2.22% | 认知输出 + 情感归因 |
| 4 | 赚了钱不敢告诉父母 | 视频号 628w，抖音 308w | 成功后的反常行为 + 代际认知差 |
| 5 | 被妈妈揭穿赚太多钱 | 视频号 1155w | 家庭博弈 + 低调智慧 |
| 6 | 给父母打多少钱才够 | 视频号 1363w | 量化孝心 + 仪式感 |

复刻规则：

- 保留核心机制，替换具体细节。
- 不直接复制已爆标题。
- 互动率 ≥ 2% 的机制优先复刻。
- 从未出现过的角度优先保留并放大。
- 已饱和方向必须找到新信息点，否则不要硬写。

### 9. 饱和度与稀缺性 Check

生成标题前先判断它属于哪个已验证系列：

| 系列 | 饱和度 | 处理方式 |
|---|---|---|
| 给妈妈打钱 | 高，已有 6 条 | 必须有新细节、新时间点或新物件，如第一次收红包、备忘录、表情包 |
| 母亲教育智慧 | 中，已有 3 条 | 可换不同教育场景，如夸奖、低调、家风、习惯 |
| 赚钱与父母认知差 | 中，已有 3 条 | 可换不同认知差，如火了、收入、职业、低调 |
| 潮汕妈妈/女人 | 低，可延展 | 优先用具体场景承接，如爱干净、招待亲戚、家风 |
| 学业/复读经历 | 低，可延展 | 适合个人逆袭和妈妈支持线 |

稀缺性判断：

- 标题高度相似：放弃或彻底重构。
- 角度类似但细节不同：寻找差异化切口。
- 完全未出现：优先保留并放大。

### 10. 数据验证后的标题倾向

爆款案例库显示，方比比标题更容易爆的入口通常不是“妈妈很辛苦”，而是：

- 一个可互动的问题：最喜欢妈妈哪一点。
- 一个反常行为：不敢给妈妈打钱、赚了钱不敢告诉父母。
- 一个无法算清的问题：给父母打多少钱才够。
- 一个超预期归因：母亲只教了一件事。
- 一个家庭内的低调反应：被妈妈揭穿赚太多钱。
- 一个有身份认同的标签：潮汕女人都很招财。

标题生成时优先找这些入口，而不是直接总结“妈妈很伟大”“妈妈很辛苦”。

### 11. 本次标题共创沉淀：什么是好的，什么是不好的

以下规则来自近期方比比情感口播标题共创，后续标题段必须优先参考。

#### 11.1 好的方向

**1. 从具体生活细节切入，不从抽象情绪切入。**

好的标题先抓具体物件、具体动作、具体数字，再带出情绪。

优先信息点：

- 红包、表情包、包包
- 300 块衣服
- 妈妈发朋友圈
- 妈妈做手术不说
- 妈妈不在家
- 妈妈发现赚太多钱 / 火了 / 粉丝多
- 一年三个日子 / 第一次 / 三个包

判断标准：标题里有没有一个观众能立刻想象出来的画面。

**2. “妈妈 + 钱”是高共鸣入口，但不能炫富。**

妈妈粉更容易接受“钱”相关标题，因为钱能反向表达孩子长大、心疼妈妈、想让妈妈少省一点。

可用方向：

- 怎么让妈妈放心花钱
- 妈妈舍不得花钱怎么办
- 妈妈买衣服怎么总嫌贵
- 妈妈 300 块衣服都嫌贵
- 赚多少钱才够养好妈妈
- 给妈妈多少钱才够
- 妈妈发现我赚太多钱后

注意：钱不是用来显摆“我有钱”，而是表达“我终于有能力照顾妈妈”。

**3. 短文案要优先用疑问式，不要直接给答案。**

短文案本身信息少，如果标题直接讲答案，会失去张力。

可用句式：

- 妈妈和工作到底哪个更重要
- 再忙真的能把妈妈放第一吗
- 忙成这样还顾得上妈妈吗
- 怎么让妈妈放心花钱
- 妈妈舍不得花钱怎么办

疑问句必须问出观众心里的真实问题，不能空泛提问。

**4. 正向情绪优先。**

方比比标题可以有冲突，但落点必须让妈妈看到后觉得温暖、被理解、被孩子放在心上。

同一事实，要优先选正向表达：

| 事实 | 不优先 | 更优先 |
|---|---|---|
| 妈妈拒绝过节安排 | 被妈妈拒绝 | 我妈母亲节也要陪她妈 |
| 妈妈不收红包 | 妈妈不要红包 | 妈妈嘴上说不用红包 |
| 妈妈不舍得花钱 | 妈妈太省了 | 怎么让妈妈放心花钱 |
| 妈妈做手术不说 | 妈妈骗我 | 妈妈别怕麻烦我 |

**5. 妈妈粉转发点通常是“孩子懂事”和“妈妈被肯定”。**

容易转发的标题不是替妈妈讲大道理，而是让妈妈看到：

- 孩子长大了
- 孩子懂妈妈
- 孩子把妈妈放在心上
- 妈妈的付出被看见
- 妈妈也可以被孩子宠

示例方向：

- 妈妈最爱的三个包
- 妈妈别怕麻烦我
- 长大后最想被妈妈需要
- 我妈养的是一整个家
- 妈妈才是家的主心骨

**6. 金句收尾要从比比角度出发。**

结尾金句不要像普适劝告，要像比比自己的真心话。

更优方向：

- 我想让妈妈习惯我的偏爱
- 我想让妈妈被我好好宠着
- 我想让妈妈放心收下爱
- 我想让妈妈开心得像个小孩

不优先方向：

- 妈妈值得被偏爱
- 妈妈也要被宠

后者可以作为方向，但最终最好改成比比视角，否则容易像旁观者说教。

**7. 标题最好保留“年入百万00后”IP锚点。**

情感口播主标题默认保留：

```text
年入百万00后 + 钩子
```

除非用户明确要求做小号、vlog 或非情感口播标题。

#### 11.2 不好的方向

**1. 不要直接讲答案。**

例如：

- 再忙也要照顾妈妈
- 妈妈永远是第一
- 工作再忙也不能忘了妈妈

这些句子适合做副标题或收尾，不适合做主标题。主标题要留缺口。

**2. 不要太像说明文。**

例如：

- 自媒体赚钱指南大公开
- 涨粉 100w 秘诀大公开
- 百万粉丝的新书推荐

这种标题像内容简介，不像点击钩子。尤其“指南”“秘籍大公开”“推荐新书”要慎用。

如果要讲书，不能直接讲书，要讲人、结果、稀缺感，例如：

- 00后老板涨粉100w后，终于把方法说出来了
- 00后老板涨粉100w，原来靠的不是运气

**3. 不要只抓文案里的漂亮金句。**

金句常常是答案，不是标题。

比如：

- 潮汕女人像水一样温柔
- 妈妈值得被偏爱
- 老妈永远是第一

这些更适合副标题、封面小字、收尾，不一定适合主标题。

**4. 不要把妈妈写得负面。**

避免：

- 妈妈骗我
- 妈妈太省了
- 妈妈不愿花钱
- 妈妈做手术都不说（可用，但要谨慎，钩子强但情绪重）

优先改成：

- 妈妈别怕麻烦我
- 妈妈舍不得花钱怎么办
- 怎么让妈妈放心花钱

**5. 不要只讲“火了”“粉丝多”。**

“100w 粉丝”“3 亿播放”“火爆全网”本身对妈妈粉吸引有限，必须和妈妈反应绑定。

更优方向：

- 100w粉丝后妈妈只说了一句话
- 火了以后妈妈还是让我低调
- 被妈妈发现赚太多钱后

#### 11.3 已验证可复用的小模型

**模型 A：具体物件 + 数字悬念**

适合红包、衣服、礼物、包包。

示例：

- 妈妈最爱的三个包
- 妈妈300块衣服都嫌贵
- 红包只是第一个包

**模型 B：妈妈嘴上拒绝 + 孩子看懂了**

适合红包、礼物、花钱、陪伴。

示例：

- 妈妈嘴上说不用红包
- 妈妈舍不得花钱怎么办
- 怎么让妈妈放心花钱

**模型 C：妈妈怕麻烦孩子 + 孩子想被需要**

适合手术、生病、报喜不报忧。

示例：

- 妈妈别怕麻烦我
- 长大后最想被妈妈需要
- 妈妈有事一定要告诉我

**模型 D：妈妈不是养孩子，是撑起家**

适合家庭责任、主心骨、妈妈不在家。

示例：

- 我妈养的是一整个家
- 妈妈不在家怎么办
- 妈妈才是家的主心骨

**模型 E：孩子优秀归因妈妈**

适合夸妈妈教育、言传身教、孩子懂事。

示例：

- 孩子优秀不是偶然的
- 好孩子是妈妈养出来的
- 妈妈把自己活成了答案

#### 11.4 主标题输出纪律

默认只输出主标题，不主动补副标题、封面小字、收尾金句。

主标题负责点击：

- 要有高情绪浓度、高信息浓度、高画面感
- 要展现过程而非结果
- 要有问题、反差、数字、动作、物件或关系缺口
- 不要把答案说完

金句 / 封面小字 / 收尾：

- 只有用户明确要求时才输出
- 可以更温情、更金句、更像答案
- 不要默认塞进标题输出里

判断：如果一句话是在解释情绪、总结答案、升华价值，它多半不是主标题。主标题要先让人点进去。

#### 11.5 标题生成前的强制自检

输出标题前，必须快速检查：

1. 这条标题是不是主标题？如果更像金句、封面小字、收尾答案，就不要当主标题输出。
2. 有没有把答案提前讲完？
3. 有没有具体物件、数字、动作或关系？
4. 妈妈粉看到后会不会觉得“这和我有关”？
5. 它是正向温暖，还是让妈妈担心、焦虑、被指责？
6. 是否保留“年入百万00后”IP锚点？
7. 如果文案很短，是否优先尝试疑问式？
8. 如果用了“书/指南/秘籍/大公开”，能不能换成人、结果和稀缺感？
9. 是否同时满足：高情绪浓度、高信息浓度、高画面感、展现过程而非结果？

### 12. 收红包类标题 Check：动作张力要有，但不能过火

以下规则来自“妈妈需要三个包 / 给妈妈发红包”标题优化。

#### 12.1 方向判断

“妈妈最爱的三个包”有包装感和记忆点，但如果要做主标题，往“妈妈不肯收红包”方向优化通常更有点击张力。

核心矛盾是：

```text
不是我想不想给，是妈妈怎么都不肯收。
```

这个方向比单纯写“红包 / 表情包 / 包包”更适合主标题，因为它同时具备：

- 动作：求、哄、收、拒收
- 冲突：孩子要给，妈妈不要
- 钱和妈妈：高共鸣入口
- 开环：为什么妈妈不收？最后怎么让她收？

#### 12.2 好的表达

优先使用有动作但不冒犯妈妈的词。

可用标题方向：

- 年入百万00后 让妈妈收钱有多难
- 年入百万00后 天天求妈妈收红包
- 年入百万00后 怎么哄妈妈收红包
- 年入百万00后 妈妈总是不收我红包
- 年入百万00后 给妈妈红包她总说不要
- 年入百万00后 妈妈拒收红包怎么办

优先词：

- 求
- 哄
- 总说不要
- 拒收
- 有多难
- 怎么让 / 怎么哄

其中“哄”最贴比比人设，因为它不是强迫妈妈，而是孩子在宠妈妈。

#### 12.3 不好的表达

以下词有张力，但容易过火或偏离比比调性：

| 表达 | 问题 |
|---|---|
| 跪求妈妈收红包 | 太重、太俗，容易像低俗标题 |
| 逼着妈妈收红包 | 关系感太硬，容易让妈妈不舒服 |
| 专治妈妈不收红包 | 像教程号 / 干货号，不像比比情感口播 |
| 老是拒收我红包 | 口语真实，但略平，需要补动作或悬念 |

#### 12.4 推荐排序

如果遇到类似“给妈妈钱 / 红包 / 礼物妈妈不收”的文案，优先按这个顺序尝试：

1. **年入百万00后 让妈妈收钱有多难**  
   最强：把红包升级成普遍难题，妈妈粉代入强。

2. **年入百万00后 天天求妈妈收红包**  
   动作强：“天天求”有张力，但不如“跪求”过火。

3. **年入百万00后 怎么哄妈妈收红包**  
   最温柔：“哄”符合比比宠妈妈的人设。

#### 12.5 标题取舍原则

原则：

- “不肯收红包”优先做主标题，负责点击。
- “三个包”更像文案里的记忆点，不要默认拆成副标题。
- 如果用户只要标题，只输出主标题，不主动补副标题。

### 13. 情绪浓度优先 Check：好坏对比

以下规则来自“妈妈做手术瞒着孩子”标题共创。后续遇到类似“妈妈怕麻烦孩子 / 报喜不报忧 / 生病不说 / 偷偷扛事”的文案，必须先用本段检查。

#### 13.1 总原则

标题第一优先级是**情绪浓度**。画面感、冲突感、信息密度都要服务于情绪浓度。

```text
情绪浓度第一
画面感、冲突感、信息密度 = 提升情绪浓度的三种抓手
```

不要只问“标题有没有信息”，要先问：

- 观众看到后有没有心里一紧？
- 妈妈粉会不会觉得“这说的不就是我吗”？
- 孩子会不会觉得“我妈也这样，我心疼”？
- 标题有没有一个没说透的关系缺口？

#### 13.2 好坏对比一：动作悬念 vs 心理悬念

| 不够好 | 更好 | 为什么 |
|---|---|---|
| 年入百万00后 妈妈瞒着我做了件事 | 年入百万00后 妈妈不敢告诉我的事 | “瞒着”是动作，只制造事件悬念；“不敢”是心理，制造关系悬念和情绪悬念 |
| 年入百万00后 妈妈做手术都瞒着我 | 年入百万00后 妈妈最不敢告诉我的事 | 前者直接泄露事件，后者先保留情绪缺口，让正文揭晓手术 |
| 年入百万00后 妈妈有事没告诉我 | 年入百万00后 妈妈有件事不敢说 | “没告诉”偏信息缺失，“不敢说”带出妈妈的怕、忍、心疼孩子 |

判断：

- “瞒着 / 没告诉”偏动作。
- “不敢 / 最不敢 / 怕我知道”偏心理。
- 方比比妈妈粉内容，优先使用能保护妈妈动机的心理词。

#### 13.3 好坏对比二：概念情绪 vs 画面情绪

| 不够好 | 更好 | 为什么 |
|---|---|---|
| 年入百万00后 妈妈别怕麻烦我 | 年入百万00后 妈妈病床上还怕耽误我 | 前者是正确情绪，后者有病床画面，情绪更落地 |
| 年入百万00后 长大后最想被妈妈需要 | 年入百万00后 妈妈一个人进了手术室 | 前者适合副标题或金句，后者适合主标题，画面冲击更强 |
| 年入百万00后 妈妈太会报喜不报忧 | 年入百万00后 妈妈手术后一周才告诉我 | 前者是概念，后者有时间和事件，更能让人心疼 |

判断：

- 概念句常常适合副标题、收尾、封面小字。
- 主标题优先用可视化画面：病床、手术室、电话、一周、红包、300块衣服、朋友圈。
- 情绪必须挂在画面上，不能只停在“懂事 / 心疼 / 需要”这些抽象词上。

#### 13.4 好坏对比三：信息密度 vs 情绪缺口

| 不够好 | 更好 | 为什么 |
|---|---|---|
| 年入百万00后 妈妈做完手术一周才说 | 年入百万00后 妈妈偷偷瞒了我一周 | 前者信息完整，后者多了“偷偷”，情绪缺口更强 |
| 年入百万00后 妈妈怕我担心瞒了手术 | 年入百万00后 妈妈最不敢告诉我的事 | 前者信息密度高，但把答案说多了；后者情绪浓度和开环更强 |
| 年入百万00后 妈妈做手术还怕我请假 | 年入百万00后 妈妈不敢耽误我工作 | 后者把妈妈心理说出来，更容易被妈妈粉代入 |

判断：

信息密度不是越满越好。信息压得太满，会提前泄露答案。更好的标题是：

```text
保留一个强信息锚点 + 留住一个情绪缺口
```

#### 13.5 可复用词库：提升情绪浓度

**隐瞒类**：

- 背着我
- 偷偷
- 瞒了我一周
- 一直没说
- 不敢告诉我
- 最不敢告诉我
- 有件事不敢说
- 怕我知道

**极限类**：

- 最
- 都
- 连
- 一周
- 一个人
- 做完才说
- 到病床上还

**心理类**：

- 不敢麻烦我
- 怕我担心
- 怕耽误我工作
- 怕我牵挂
- 不敢开口需要我

**画面类**：

- 病床
- 手术室
- 医院
- 电话
- 一周后
- 红包
- 表情包
- 300块衣服
- 朋友圈

#### 13.6 三轨延展法

遇到标题卡住时，不要只出一组。必须按三轨各延展至少 5 个：

1. **画面感轨**：把场景打出来。
   - 例：妈妈一个人进了手术室
   - 例：妈妈病床上还说没事

2. **情绪浓度轨**：用隐瞒、悬念、极限、数字拉情绪缺口。
   - 例：妈妈最不敢告诉我的事
   - 例：妈妈偷偷瞒了我一周

3. **信息密度轨**：压入事件、动作、时间、关系。
   - 例：妈妈做完手术一周才说
   - 例：妈妈怕我担心瞒了手术

最后排序时，优先选**情绪浓度最高且开环成立**的标题，而不是信息最完整的标题。

#### 13.7 本案例推荐排序校准

同一条文案里，标题大致按这个优先级判断：

1. **年入百万00后 妈妈最不敢告诉我的事**  
   情绪浓度最高，“最 + 不敢 + 事”形成心理悬念，保护妈妈动机，不泄露手术。

2. **年入百万00后 妈妈偷偷瞒了我一周**  
   情绪浓度 + 数字都强，“偷偷 + 一周”让隐瞒变具体。

3. **年入百万00后 妈妈病床上还怕耽误我**  
   画面最强，适合封面或副钩子；如果主标题想强画面可用。

4. **年入百万00后 妈妈做完手术一周才说**  
   信息密度最高，但情绪缺口不如前面几条。

5. **年入百万00后 长大后最想被妈妈需要**  
   金句感强，但更适合副标题或收尾，不优先做主标题。

### 14. 家庭主心骨类标题 Check：抓“求妈妈回家”的过程动作

以下规则来自“家里所有事情都是妈妈说了算 / 妈妈不在家就打电话问什么时候回来”的标题共创。

#### 14.1 总原则

这类文案不要只总结“妈妈很重要”“妈妈撑起家”，要抓家庭运转里的具体过程。

核心结构是：

```text
妈妈不在家 → 全家运转不了 → 打电话求妈妈回来
```

这比单纯说“妈妈是主心骨”更有画面冲突，因为它让观众看见一家人离不开妈妈的现场。

#### 14.2 好坏对比

| 不够好 | 更好 | 为什么 |
|---|---|---|
| 年入百万00后 妈妈不在家怎么办 | 年入百万00后 全家求着妈妈回家 | 前者只是问题，后者有动作、人物关系和画面冲突 |
| 年入百万00后 我妈养的是一整个家 | 年入百万00后 全家求着妈妈回家 | 前者情绪厚但偏总结，后者展现过程，更像主标题 |
| 年入百万00后 妈妈才是家的主心骨 | 年入百万00后 妈妈一走全家都慌了 | 前者是结果判断，后者是现场反应 |
| 年入百万00后 家里少不了妈妈 | 年入百万00后 妈妈不在家只能打电话 | 前者正确但平，后者有具体动作 |
| 年入百万00后 家里什么都等妈妈定 | 年入百万00后 妈妈不在家谁说了算 | 前者信息密度强，后者更有冲突缺口 |

#### 14.3 三维度拆法

遇到“妈妈撑起家 / 妈妈管全家 / 妈妈不在家”的文案，先按三维度各出标题：

**情绪浓度轨**：妈妈付出一辈子，孩子怎么还都不够。

- 年入百万00后 我妈养的是一整个家
- 年入百万00后 妈妈一辈子都在撑家
- 年入百万00后 我欠妈妈的是一个家

**信息密度轨**：一家四口、所有事情、吃什么、亲戚怎么招待都靠妈妈。

- 年入百万00后 家里所有事都听我妈
- 年入百万00后 我妈一个人管一家四口
- 年入百万00后 家里什么都等妈妈定

**画面冲突轨**：妈妈一不在家，全家就打电话找她。

- 年入百万00后 全家求着妈妈回家
- 年入百万00后 妈妈一走全家都慌了
- 年入百万00后 妈妈不在家只能打电话
- 年入百万00后 妈妈不在家谁说了算

排序时优先选：**画面冲突强 + 情绪浓度不掉 + 开环成立** 的标题。

#### 14.4 推荐排序校准

本类文案优先级大致如下：

1. **年入百万00后 全家求着妈妈回家**  
   最强：有“全家 + 求着 + 回家”的动作现场，既有情绪也有冲突。

2. **年入百万00后 妈妈一走全家都慌了**  
   冲突强：“一走 / 都慌”有极限感，但略偏戏剧化。

3. **年入百万00后 妈妈不在家只能打电话**  
   画面具体，能接原文“妈什么时候回来”。

4. **年入百万00后 妈妈不在家谁说了算**  
   问题感强，适合突出“家里所有事妈妈说了算”。

5. **年入百万00后 我妈养的是一整个家**  
   情绪厚，但偏结果判断；更适合在正文里作为金句，不优先当主标题。

#### 14.5 关键提醒

- “主心骨 / 撑起家 / 养一整个家”是价值判断，情绪厚，但容易像结论。
- “求着妈妈回家 / 打电话问妈妈什么时候回来 / 妈妈一走全家慌”是过程动作，更适合标题。
- 顶级原则是：展现过程而非结果。先让观众看见家里离不开妈妈，再让正文讲妈妈养的是一整个家。
